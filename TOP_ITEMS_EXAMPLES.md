## 7. ตัวอย่างการใช้งานจริง

### ตัวอย่างที่ 1: คำถามแบบธรรมชาติ
**คำถาม:** "อะไรขายดีที่สุด 6 เดือน"
**Output:**
```json
{
  "action": "top_items",
  "time_period": {
    "period": "month",
    "value": 6
  },
  "limit": 10,
  "text": "อะไรขายดีที่สุด 6 เดือน",
  "session_id": "u_001"
}
```

### ตัวอย่างที่ 2: คำถามแบบระบุจำนวน
**คำถาม:** "TOP 20 สินค้า 2 เดือน"
**Output:**
```json
{
  "action": "top_items",
  "time_period": {
    "period": "month",
    "value": 2
  },
  "limit": 20,
  "text": "TOP 20 สินค้า 2 เดือน",
  "session_id": "u_001"
}
```

### ตัวอย่างที่ 3: คำถามแบบไม่ระบุเวลา
**คำถาม:** "สินค้าขายดีที่สุด"
**Output:**
```json
{
  "action": "top_items",
  "time_period": null,
  "limit": 10,
  "text": "สินค้าขายดีที่สุด",
  "session_id": "u_001"
}
```

### ตัวอย่างที่ 4: คำถามแบบทั้งหมด
**คำถาม:** "สินค้าทั้งหมดที่ขายดี"
**Output:**
```json
{
  "action": "top_items",
  "time_period": null,
  "limit": 100,
  "text": "สินค้าทั้งหมดที่ขายดี",
  "session_id": "u_001"
}
```

## 9. Default Values

- **Limit:** 10 (ถ้าไม่ระบุจำนวน)
- **Limit (All):** 100 (ถ้ามีคำว่า "ทั้งหมด", "ALL", "ทุก", "EVERY")
- **Time Period:** null (ถ้าไม่ระบุเวลา)

---

## 10. หมายเหตุ

- Parser รองรับทั้งภาษาไทยและอังกฤษ
- Parser รองรับคำถามแบบธรรมชาติ (ไม่ต้องระบุจำนวน)
- Parser extract ข้อมูลอัตโนมัติ (limit, time_period)
- Parser รองรับรูปแบบการถามที่หลากหลาย
- Parser รองรับการถามแบบย่อ (เช่น "ขายดี", "TOP")
